# Renewable Energy Management Portfolio

This is a portfolio website for the Renewable Energy Management (REM) coursework assignment.

## Website Structure

```
rem-portfolio/
├── index.html          # Main HTML file
├── styles.css          # Dark mode styling
├── script.js           # Quiz and interactions
├── data/               # Your coursework documents
│   ├── assignments/    # PDFs of assignments
│   ├── case-studies/   # Case study documents
│   ├── posters/        # Poster PDFs
│   ├── presentations/  # PowerPoint files
│   └── research/       # Research papers
└── README.md          # This file
```

## How to Add Your Documents

1. **Assignments**: Place PDF files in `data/assignments/`
   - Name them: `solar-pv-design.pdf`, `wind-assessment.pdf`, etc.

2. **Case Studies**: Place PDF files in `data/case-studies/`
   - Name them: `bhadla-solar.pdf`, `offshore-wind-europe.pdf`, etc.

3. **Posters**: Place PDF files in `data/posters/`
   - Name them: `solar-future.pdf`, `energy-transition.pdf`, etc.

4. **Presentations**: Place PowerPoint files in `data/presentations/`
   - Name them: `solar-pv.pptx`, `wind-economics.pptx`, etc.

5. **Research**: Place PDF files in `data/research/`
   - Name them: `perovskite-cells.pdf`, `green-hydrogen.pdf`, etc.

## GitHub Pages Deployment

1. Push all files to a GitHub repository
2. Go to Settings → Pages
3. Select source: Deploy from a branch
4. Choose branch: main
5. Your site will be live at `https://yourusername.github.io/rem-portfolio`

## Features

- Dark mode attractive design
- Fully responsive (mobile-friendly)
- Interactive quiz with 10 REM questions
- Expandable work showcase sections
- Smooth scrolling navigation
- Contact form
- All coursework documents downloadable

## Customization

You can edit the content in `index.html` to:
- Update the About Me section
- Modify study cards
- Add/remove work items
- Change contact information

Edit `styles.css` to:
- Change colors (look for CSS variables at the top)
- Modify spacing
- Adjust fonts

Edit `script.js` to:
- Add more quiz questions
- Change quiz behavior
- Modify animations
